# JSDoc Conformance Fixtures

These fixtures test oxfmt's JSDoc formatting against [prettier-plugin-jsdoc](https://github.com/hosseinmd/prettier-plugin-jsdoc).

## Structure

Each fixture consists of:

- `<name>.js` / `<name>.ts` — input file
- `<name>.output.js` / `<name>.output.ts` — expected output (generated by running prettier-plugin-jsdoc)
- `options.json` (optional, per-file or per-directory) — plugin options override

## Origin

Expected outputs were generated using **prettier-plugin-jsdoc** at commit [`1bbf9ee`](https://github.com/hosseinmd/prettier-plugin-jsdoc/tree/1bbf9ee43e31f054cc27b7355b4dc53a90b31302) with default options unless an `options.json` is present.

Default options: `jsdocCapitalizeDescription: true`, `jsdocSingleLineComment: true`, `printWidth: 80`.

Upstream snapshot reference: [`tests/__snapshots__/main.test.ts.snap`](https://github.com/hosseinmd/prettier-plugin-jsdoc/blob/1bbf9ee43e31f054cc27b7355b4dc53a90b31302/tests/__snapshots__/main.test.ts.snap)
